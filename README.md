# Online Food Ordering System (Backend Simulation)

## Project Description
This is a backend simulation of an Online Food Ordering System developed for the SCD Lab Project. It demonstrates the use of Software Design Patterns to solve real-world problems.

## Design Patterns Used
1. **Composite Pattern:** For Menu Structure (Categories & Items).
2. **Observer Pattern:** For notifying Restaurant & Driver on new orders.
3. **Command Pattern:** For handling order placement requests.
4. **Facade Pattern:** To simplify the user interface.

## How to Run
1. Install Python 3.x.
2. Open terminal/cmd in the project folder.
3. Run the command:
   python main.py
